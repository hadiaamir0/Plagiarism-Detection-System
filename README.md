# Plagiarism Detection System

## Overview

The Plagiarism Detection System is a C++-based project developed as part of the Design and Analysis of Algorithms (DAA) course. The system analyzes textual documents and identifies similarities using Longest Common Subsequence (LCS) and Hashing techniques.

The project demonstrates algorithmic problem-solving skills and applies dynamic programming concepts to support academic integrity analysis.

## Algorithms Implemented

### Longest Common Subsequence (LCS)

The LCS algorithm identifies the longest matching sequence between two documents and is used to calculate similarity scores.

### Hashing

Hashing converts document contents into numerical representations that enable efficient comparison and analysis.
