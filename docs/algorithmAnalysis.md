# Algorithm Analysis

## Longest Common Subsequence (LCS)

The LCS algorithm identifies the longest sequence of characters appearing in both documents while maintaining order.

### Complexity

Time Complexity:
O(n × m)

Space Complexity:
O(n × m)

Where:
- n = length of document 1
- m = length of document 2

## Hashing

Hashing converts document contents into numerical representations for efficient comparison and duplicate detection.

### Complexity

Time Complexity:
O(n)

Space Complexity:
O(1)

## Combined Approach

The system combines LCS and Hashing to improve similarity analysis accuracy and demonstrate multiple algorithmic techniques used in plagiarism detection.
